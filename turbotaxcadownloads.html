How do I download my software from TurboTax.com?

Solved • turbotax.ca/download • 2119 • By turbotax

For TurboTax Online customers, no software download is needed.

For TurboTax CD/Download purchases, once you’ve received an order confirmation for your purchase,
follow these instructions to download your TurboTax software:

  1. Go to Access my downloads and sign in with the same user ID you used to purchase your software
  2. On Your TurboTax account, select the Downloads tab
  3. Select the version you want to download and save it somewhere on your computer where you can find it
    • If you need help finding your download, we have instructions for Windows and Mac
  4. Install your software by double-clicking the downloaded file
    • If you need help installing, we have instructions for Windows and Mac

IMPORTANT INFORMATION

If you don’t see your product in the Downloads tab, go to the Intuit account recovery page and enter your email address to find other user IDs associated with your email. Your download might be in one of your other accounts.


How to switch between Mac and PC/Windows with... a CD or Download
  1. Go to Access my downloads and sign in
  2. Select Downloads
  3. Get the license code
    • Your license code is a 16-character combinations of letters and numbers (XXXX-XXXX-XXXX-XXXX) that verifies the software and allows you to finish installing and begin using TurboTax. See instructions for where to find your license code
  4. Scroll down the screen until you see Did you order a TurboTax product that you don’t see listed above? then select Enter your license code here
  5. Enter your license code from your CD or download
  6. Under Which operating system do you use?, select the operating system for the version you wish to install, either PC/Windows or Mac
  7. Select Get Download to download your new version

If you want to switch from TurboTax Online to TurboTax CD/Download, follow these instructions.

How to install turboTax

  • Click the link for the TurboTax product that you purchased.
    • The installer is downloaded to your computer, and the installer window appears on the screen.
  • Follow the instructions on the screen, clicking Next to move through the process
  • On the Ready to Install screen, select Install .
  • Once the installation is complete, select Launch TurboTax .
    • The Enter Licence code window appears.
  • On the Ready to Install screen, select Install.
